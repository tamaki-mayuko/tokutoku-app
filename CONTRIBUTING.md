# 徳徳ネットを育てる(参加のしかた)

徳徳ネットは、ひとりの創始者のものではなく、
理念に共感するみなさんで育てるプロジェクトです。
プログラミングができなくても、できることがたくさんあります。

## だれでもできること

- **使ってみて感想を伝える** … Issues(またはメール)で
  「ここで指が止まった」「この言葉が分かりにくい」を教えてください。
  高齢のご家族の反応は、最高の贈り物です。
- **地域で試す** … README の手順で、あなたの町の徳徳ネットを
  立ててみてください。導入会の様子や学びを共有してもらえたら、
  他の地域の宝になります。
- **つなぐ** … 社会福祉協議会、地域包括支援センター、老人会、
  民生委員、シビックテック団体(Code for X など)との橋渡し。

## 開発・デザインで手伝えること(募集中の宿題)

- 導入会スタッフ用の会員管理画面(現在はSupabase管理画面を直接操作)
- 「座をひらく」機能の実装
- 本人認証の強化(デジタル認証アプリ連携/Supabase Auth)
- 音声読み上げ・より高いアクセシビリティ
- 通知のLINE連携(家族への見守り通知)
- 多言語化(やさしい日本語を含む)

## お願い(変更の作法)

- 変更を提案するときは、PHILOSOPHY.md に照らして
  「どの条項に資するか」を一言添えてください。
- 特に第一条(徳はお金で買えない)・第二条(入力は2つだけ)・
  第四条(金銭カテゴリを作らない)に反する変更は受け入れられません。
- 機能を足す提案と同じくらい、引く提案を歓迎します。

## 行動規範

このプロジェクトの目的は高齢者の孤立を防ぐことです。
議論においても、経験や技術力にかかわらず、互いに
「喜んでもらえること、たまに褒められること、仲間ができること」
を大切にしてください。
